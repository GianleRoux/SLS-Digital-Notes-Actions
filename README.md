# SLS Digital Notes & Actions

Shared register for notes, action items, and decisions between Gian & Nic.

---

## How to use

- **Issues** → Action items (assign, label, close when done)
- **`notes/` folder** → Meeting notes, decisions, context (Markdown files)
- **Labels** → `action`, `note`, `decision`, `review`, `blocked`

This repo is separate from all SLS Digital source code repositories.
